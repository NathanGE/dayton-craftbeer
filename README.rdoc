== README

Dayton-Craftbeer is a simple web application to help people find microbreweries and taprooms in Dayton, Ohio when they want a good beer and to promote these wonderul small businesses.

This application was built with Ruby on Rails:

* Ruby 2.0

* Rails 4.0

* Visit Dayton-Craftbeer live at https://dayton-craftbeer.herokuapp.com/
<tt>rake doc:app</tt>.
